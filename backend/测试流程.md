测试步骤

1\. 填写数据库配置

在db\_config.json 中填写自己的数据库密码

2\. 启动：

bashcd D:\\study\\code\\aiproject\\aiproject\\backend

uvicorn app.main:app --reload --port 8000

3\. 打开 localhost:8000/docs，依次调：

POST /api/auth/anonymous → 拿到 session\_id

PUT /api/user/config → 填 X-Session-ID + 请求体 {"api\_key": "你的Key", "model": "deepseek-chat"}

POST /api/user/config/test → 确认 Key 可用

POST /api/papers/upload → 选一个 PDF，拿到 paper\_id

等 30-90 秒 → GET /api/papers/{paper\_id} → 检查 parse\_status

如果是 completed → POST /api/qa/{paper\_id} → 提一个问题

POST /api/reports/{paper\_id} → {"report\_type": "quick"}
